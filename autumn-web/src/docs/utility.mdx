# Utility

General-purpose commands for help, diagnostics, and command discovery.

Autumn supports both prefix and slash commands. Examples below use prefix syntax.

## `!help`
Shows available commands by page or category.

**Usage:**
```bash
!help [page|category]
```

**Examples:**
```bash
!help
!help 2
!help utility
```

---

## `!usage`
Shows syntax for a specific command.

**Usage:**
```bash
!usage <command>
```

**Examples:**
```bash
!usage ban
!usage timeout
!usage modlogchannel
```

---

## `!ping`
Checks bot responsiveness/latency.

**Usage:**
```bash
!ping
```

---

## `!pagetest`
Developer utility for pagination behavior testing.

**Usage:**
```bash
!pagetest [page]
```

---

## `!universe`
Lightweight easter-egg style command.

**Usage:**
```bash
!universe
```
